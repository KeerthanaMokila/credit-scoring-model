# 💳 Credit Scoring Prediction System

This is a Machine Learning web application that predicts whether a loan applicant is **High Risk or Low Risk** based on financial and personal details.

The model is trained using **Random Forest Classifier** and deployed using **Streamlit**.

---

## 🚀 Live App
https://credit-scoring-model-jtknwbdgzzv2cjhtbwlbmg.streamlit.app/

---

## 📊 Features
- Predicts credit risk (Good / Bad)
- User-friendly web interface
- Real-time prediction
- Probability output
- Trained ML model using real dataset

---

## 🧠 Machine Learning Workflow
- Data Cleaning
- Handling Missing Values
- Label Encoding (categorical features)
- Model Training (Random Forest & Logistic Regression)
- Model Evaluation
- Deployment using Streamlit
